# MarketSim

MarketSim is a stock‑market simulation platform that lets new users practice trading with virtual money. It provides a safe, educational environment where people can explore how markets work, try different strategies, and learn without financial risk.

## Features

- Secure user accounts with profile settings and password recovery  
- Real‑time and historical market data with charts and indicators  
- Virtual trading for buying, selling, and managing positions  
- Portfolio summaries showing performance and holdings  
- Watchlists for tracking stocks of interest  
- Achievements that reward user activity  
- Optional advanced tools like stop‑losses, limit orders, short selling, and options trading  

## Purpose

MarketSim helps beginners understand market behavior, build confidence, and develop trading skills through hands‑on experience using simulated cash.
